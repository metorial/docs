## Start With Workforce

Metorial helps teams replace scattered tool access, one-off MCP setup, and hard-to-audit credentials with a governed Workforce. Admins decide which tools and skills are approved, employees discover them through portals, agents connect through managed MCP access, and activity logs show how access is used.
