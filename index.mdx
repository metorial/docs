## Introduction

Metorial is agentic infrastructure for connecting AI agents, products, MCP clients, and CLIs to the tools users authorize.

Use Metorial when you need AI systems to call real tools with the right user-approved access, without building every OAuth flow, MCP server, credential store, and activity trail yourself.

Users connect tools through portals. AI clients connect through standard MCP URLs. Teams control which integrations and skills are available, then review traces, sessions, tool calls, auth events, and failures after work runs.

Workforce is the admin control plane. Portals are where users authorize integrations and discover skills. Magic MCP turns that approved access into a standard connection URL for AI clients, agents, MCP clients, and CLIs.
