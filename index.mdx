## Introduction

Metorial is a set of controls, hosted surfaces, and APIs for connecting AI systems to the apps and internal systems employees use every day.

Use Metorial to approve integrations, publish reusable skills, give MCP clients a managed connection URL, and monitor access events, tool calls, and usage from one place.

Workforce is the main product surface. Admins decide which integrations and skills are approved, employees connect through portals, and AI clients or agents use that approved access through managed MCP connections.

## Common Use Cases

| Use case | What Metorial enables |
| --- | --- |
| Employee AI assistants | Let employees connect approved apps such as GitHub or Linear, then use those connections from AI clients and agents. |
| Internal agent platforms | Give agents governed MCP access to the integrations and skills they need, with activity logs for every session and tool call. |
| Customer-facing AI features | Publish portals where customers or partners can connect approved resources before your AI product acts on their behalf. |
| Reusable workflows | Package repeatable work as skills so teams can share governed workflows instead of rebuilding prompts and tool setup each time. |
