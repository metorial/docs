## Introduction

Metorial is agentic infrastructure for connecting AI agents, products, MCP clients, and CLIs to the tools users authorize.

Use Metorial to give every employee, agent, and AI client one place to connect integrations and skills, with governance, access control, and tracing built in.

Users connect tools through portals. AI clients connect through standard MCP URLs. Teams can review sessions, tool calls, auth events, and failures after work runs.

## Common Use Cases

| Use case | What Metorial enables |
| --- | --- |
| AI products | Connect your product to the tools each user authorizes. |
| Employee AI clients | Let employees connect integrations once and reuse them across AI clients. |
| Internal agents | Give company-run agents governed access to approved integrations and skills. |
| MCP clients and CLIs | Provide a standard MCP URL for tool access without custom OAuth or one-off servers. |
| Custom providers | Bring internal APIs, private tools, and remote MCP servers into the same governed runtime. |
