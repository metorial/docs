Managed providers let you deploy custom MCP server code without managing infrastructure. Upload your code and Metorial handles hosting, scaling, and deployment.

## Supported Languages

Metorial supports two languages for custom providers:

<Info>
**TypeScript/JavaScript (Node.js)**
- Full npm package support
- Automatic TypeScript compilation
- Node.js 18+ runtime
</Info>

<Info>
**Python**
- Standard Python packages with pip
- Python 3.10+ runtime
- Support for async/await
</Info>

## Creating a Custom Provider

<Steps>
  <Step title="Navigate to Custom Providers">
    Go to **Gateway** → **Custom Providers**.

    ![Navigate to custom providers](/images/custom-servers/navigate-managed-servers.png)
  </Step>
  <Step title="Create Provider">
    Click **Create Custom Provider** and select **Custom Provider**.

    ![Create custom provider](/images/custom-servers/create-managed-server.png)
  </Step>
  <Step title="Upload Code">
    Upload your MCP server code.
  </Step>
  <Step title="Deploy">
    Click **Deploy** and wait for the build to complete.
  </Step>
</Steps>
