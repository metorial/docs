## Testing a provider

Providers in Metorial expose capabilities such as tools and resources to AI assistants. Explorer gives you a dashboard surface for checking those capabilities.

<Steps>
  <Step title="Open the Explorer">
    Open **Explorer** from the project sidebar or click **Open Explorer** from a provider page.

    ![Explorer](/images/dashboard-audit/dashboard-explorer.png)
  </Step>
  <Step title="Choose a Provider Context">
    Select the provider, integration, auth config, or Magic MCP context you want to test. If the provider needs auth, create or select auth credentials first.

    ![Linear provider tools](/images/dashboard-audit/provider-linear-capabilities.png)
  </Step>
  <Step title="Test Server Capabilities">
    Select a tool, enter sample input, and run it. Start with read-only tools before testing write or destructive tools.

    ![Session Viewer Interface](/images/metorial-101/testing-mcp-servers/step3.png)
  </Step>
  <Step title="View Result">
    View the resulting response from the provider to see the data your AI integrations will consume.

    ![MCP Message Stream](/images/metorial-101/testing-mcp-servers/step4.png)
  </Step>
</Steps>
