## Configure a provider

Providers in Metorial provide tools and resources to AI assistants through the Model Context Protocol.

Provider pages now expose two setup paths:

- **Integration**: reusable provider setup for sessions and applications
- **Magic MCP Server**: a managed MCP endpoint you can connect to clients and agents

<Steps>
  <Step title="Open the Provider Catalog">
    Navigate to **Providers** in the dashboard and choose a provider such as **GitHub** or **Linear**.

    ![Provider catalog](/images/dashboard-audit/dashboard-providers-catalog.png)
  </Step>
  <Step title="Review the Provider">
    The provider page includes an overview, tools and capabilities, versions, and setup actions.

    ![GitHub provider page](/images/dashboard-audit/provider-github-detail-confirmed.png)
  </Step>
  <Step title="Choose How to Use the Provider">
    Click **Use Provider**. Choose **Integration** when you want reusable provider setup for sessions, or **Magic MCP Server** when you want a connectable MCP server endpoint.

    ![Use provider menu](/images/dashboard-audit/provider-linear-use-provider-modal.png)
  </Step>
  <Step title="Configure Auth and Tool Access">
    Select the auth method, choose or create auth credentials, and review tool filters. Tool filters are grouped as read-only, write, and destructive tools.

    ![GitHub integration setup](/images/dashboard-audit/provider-github-create-integration-github-com-selected.png)
  </Step>
  <Step title="Optional: Create a Magic MCP Server">
    Magic MCP setup asks for an auth config and tool filters, then creates a server clients can connect to.

    ![Create a Magic MCP server from GitHub](/images/dashboard-audit/provider-github-create-magic-mcp-flow.png)
  </Step>
</Steps>
